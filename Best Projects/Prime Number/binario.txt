Start program

Display a message requesting the input of a natural number
Read input
    If the input is invalid,
    Restart program

    If it is valid,
    Continue program

To check if the number is prime:
Check the remainder of successive divisions between
"input - 1" and 2.
    If there is any remainder equal to 0,
    it is not prime.

    Otherwise,
    it is prime.

Display a message stating whether the number is prime or not.
