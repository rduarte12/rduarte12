Start program

Display a message asking for input of the total number
of scores to be recorded.
    Read input
        If less than or equal to zero
        display "Invalid Number"
        restart program

        Else, continue the program normally.

    Record the input as the number of "arrays"
    of floating-point "scores".

In a "for" loop, record all the scores in
their respective "array".

Calculate the arithmetic mean of the scores.

Display a message responding to the user with the value.
